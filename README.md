# Machine-learning
Machine Learning – Course Work

A collection of assignments, experiments, and notebooks from the Machine Learning course. The work covers both classical machine learning with scikit-learn and deep learning with PyTorch.

Content

Weeks 9–13:
Probability, gradient descent, regression, classification, regularization, decision trees, SVM, PCA, and clustering.

Weeks 14–16:
Neural networks, backpropagation, image classification, CNNs, and chatbot basics.

Week 17:
Final assignments and internship (LIA) preparation.

The material is based on the course content and Hands-On Machine Learning.
